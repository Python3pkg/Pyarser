Pyarser

An HTML Parser Package written for Python

Pyarser enables users to harvest HTML data in a simple straight forward way, making that data available in a variety of data structures. It's a great addition to your python scripts. Pyarser requires the requests module to be installed.

Please see the wiki for complete API documentation.

Find us on PyPi

Pyarser supports several different classes of processing for HTML text:

  • Openers : Different ways to open the HTML string from a link.

  • Modifiers : Removes certain characters or deleting sections of HTML.

  • Harvestors : Harvests text and HTML content into Python data Structures.

  • Filters : Filters out content based on thresholds, such as text only containing numbers, or no numbers etc.

Installation

To Install, please use PIP pip install requests ~~~If you haven't already installed the requests module. pip install Pyarser

Link Functions

These functions interact directly with a given link, for fast, easy web searching and scraping.

get_site_HTML(link):

  • returns a string of HTML content from a link.

get_site_text(link):

  • returns a string of text from a link. This string has all the HTML tags <> removed, along with there contents.

search_by_phrase(phrase, link):

  • returns the fragments of text from a link that contain the continuous string phrase.

search_for_words(words, link):

  • returns the fragments of text from a link that contain ANY of the strings in words.

word_count(link):

  • counts the number of text words from a link.

get_HTML_tags(link):

  • returns a list of the tags used in an HTML document from a link.

HTML_to_TXT(link, name):

  • writes a TXT file with the text content from a link. All HTML brackets and tags are moved.
